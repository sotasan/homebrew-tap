# Sotasan Tap

## How do I install these formulae?

`brew install sotasan/tap/<formula>`

Or `brew tap sotasan/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sotasan/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
